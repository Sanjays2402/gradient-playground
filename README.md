# 🎨 Gradient Playground

An interactive CSS gradient builder with live preview, presets, and code export. Built with React, Vite, and Tailwind CSS.

![Gradient Playground](https://img.shields.io/badge/react-19-blue?style=flat-square) ![Vite](https://img.shields.io/badge/vite-8-purple?style=flat-square) ![Tailwind](https://img.shields.io/badge/tailwind-4-cyan?style=flat-square) ![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)

## ✨ Features

- 🖼️ **Full-screen live preview** — gradient takes up the entire viewport
- 🎚️ **Drag-to-position color stops** — click and drag on the gradient bar
- 🔄 **3 gradient types** — linear, radial, and conic with smooth switching
- 🧭 **Angle & position controls** — interactive dial + slider for precise control
- 📋 **One-click CSS copy** — syntax-highlighted output, ready to paste
- 🎭 **14 curated presets** — Aurora, Neon Pulse, Cosmic Fusion, and more
- 🎲 **Randomize** — instant inspiration with random gradients
- 🖤 **Dark glassmorphism UI** — frosted glass controls overlaying the gradient
- 📸 **Export as PNG** — download 1920×1080 gradient images
- ⚡ **Smooth animations** — Framer Motion throughout
- ➕ **Add/remove stops** — up to 8 color stops per gradient

## 🚀 Getting Started

```bash
# Clone
git clone https://github.com/Sanjays2402/gradient-playground.git
cd gradient-playground

# Install
npm install

# Dev server
npm run dev

# Build
npm run build
```

## 📐 Design Philosophy

Inspired by tools like Raycast and Linear — minimal, dark, and polished. The gradient preview is the hero, with controls floating in a frosted glass panel that stays out of the way.

- **No visual clutter** — subtle grid overlay, corner marks, grain texture
- **Glassmorphism controls** — backdrop-blur panel with precise opacity layers
- **Micro-interactions** — spring animations, hover states, smooth transitions
- **Typography** — Inter for UI, JetBrains Mono for code
